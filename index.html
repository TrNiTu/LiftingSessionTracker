About This Program

Lifting Session Tracker is a program that users will be able to use to record their workouts. Starting a new "lifting session" will grant users access to adding exercises to each one with the date automatically labeled. Exercises will have input options of a name, sets, reps (repetitions), weight, and an optional note. This will help users keep track of their weight lifting journey while also showing progression!
